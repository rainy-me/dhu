# @dhu/cli

```
dhu/0.1.5

Usage:
  $ dhu

Commands:
                  Log logo
  login           Save login info to local data path
  logout          Remove login info from local data path
  gpa             Get GPA
  atte            Get attendance
  info            Get info
  fs              Get fs
  task            Get tasks
  matl            Get Materials
  config [...kv]  Set config
  timetable       Download timetable csv

For more info, run any command with the `--help` flag:
  $ dhu --help
  $ dhu login --help
  $ dhu logout --help
  $ dhu gpa --help
  $ dhu atte --help
  $ dhu info --help
  $ dhu fs --help
  $ dhu task --help
  $ dhu matl --help
  $ dhu config --help
  $ dhu timetable --help
```
